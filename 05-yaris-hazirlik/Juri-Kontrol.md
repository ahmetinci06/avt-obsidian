# 👨‍⚖️ Jüri Kontrol Noktaları
#yarış #kontrol

> [!warning] Sık Kalma Noktaları
> Geçmiş yıllarda takımların en çok kaldığı maddeler 🔴 ile işaretli.

## Teknik Kontrol Sırası

### 1. Doküman Kontrolü
- [ ] Tüm dokümanlar zamanında teslim edildi
- [ ] Teknik Tasarım Raporu eksiksiz
- [ ] 🔴 Yerlilik beyanı + kanıtları hazır

### 2. Güvenlik Kontrolü
- [ ] 🔴 Acil durdurma butonları (2×) çalışıyor
- [ ] 🔴 Rollbar ölçümü (baş üstü boşluk)
- [ ] Emniyet kemeri sağlamlığı
- [ ] Yangın söndürücü
- [ ] 🔴 HV izolasyon testi (multimetre ile yerinde ölçüm)
- [ ] HV kablo rengi (turuncu)

### 3. Fren Testi
- [ ] 🔴 Eğimde durma (kayma yok)
- [ ] 🔴 Acil frenleme mesafesi
- [ ] Fren sıvısı seviyesi

### 4. Elektrik Kontrolü
- [ ] Batarya kapasitesi doğrulaması (< 3 kWh)
- [ ] BMS çalışıyor
- [ ] Sigorta/koruma sistemi
- [ ] 🔴 Pre-charge devresi

### 5. Genel Kontrol
- [ ] Araç numarası/etiket gövdede
- [ ] Takım logosu
- [ ] Sürücü belgesi
- [ ] Ağırlık tartımı

## Sık Yapılan Hatalar
1. Yerlilik beyanı eksik/yetersiz
2. Acil durdurma butonu dışarıdan erişilemiyor
3. HV kablolar turuncu değil
4. Rollbar yüksekliği yetersiz
5. Fren testi başarısız (hava kaçağı)
6. İzolasyon testi başarısız (nem/kir)
