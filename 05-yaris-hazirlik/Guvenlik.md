# 🛡️ Güvenlik Checklist
#yarış #güvenlik #critical

> [!danger] Bu liste yarış öncesi teknik kontrolde kontrol edilecek. Eksik = YARIŞA ALINMAZSINIZ.

## Acil Durdurma
- [ ] Pilot erişimli acil durdurma butonu (mushroom, kırmızı)
- [ ] Dışarıdan erişilebilir acil durdurma butonu
- [ ] Her iki buton test edildi (tüm güç kesildi)
- [ ] Butonlar "EMERGENCY STOP" / "ACİL DURDURMA" etiketi

## Yapısal Güvenlik
- [ ] Rollbar monte edildi
- [ ] Rollbar üstü ↔ sürücü başı arası > 5cm
- [ ] Rollbar dayanıklılık hesabı/testi
- [ ] Yan darbe koruması
- [ ] Ön darbe koruması (crumple zone veya tampon)

## Sürücü Güvenliği
- [ ] Emniyet kemeri (4 veya 5 nokta)
- [ ] Kemer bağlantı noktaları sağlam (şasiye boltlu)
- [ ] Kask (yarış sırasında zorunlu olabilir)
- [ ] Sürücü çıkış testi (<10 saniyede çıkabilmeli)

## Yangın Güvenliği
- [ ] Yangın söndürücü (ABC sınıfı, min 1 kg)
- [ ] Söndürücü sabit montajlı (sürücü erişiminde)
- [ ] Batarya kasası yanmaz malzeme

## Elektrik Güvenliği
- [ ] HV kablolar turuncu renk
- [ ] HV kablolar koruyucu kılıf içinde
- [ ] HV izolasyon testi: > 500Ω/V (multimetre ile)
- [ ] Ana sigorta (batarya çıkışında)
- [ ] Pre-charge devresi (inrush current koruması)
- [ ] Batarya kasası IP korumalı (en az IP44)
- [ ] Batarya kasası darbe dayanımlı (düşme testi)
- [ ] Topraklama (şasi ↔ LV GND)

## Fren Güvenliği
- [ ] Hidrolik fren sistemi çalışıyor
- [ ] Her tekerlek bağımsız kilitleyebiliyor
- [ ] Eğimde durma testi (%20 eğimde kayma yok)
- [ ] Fren sıvısı seviyesi kontrol
- [ ] Elektronik fren sinyali (PC5) çalışıyor → motor durur

## Görüş & Sinyalizasyon
- [ ] Yan aynalar (2×) veya kamera sistemi
- [ ] Ön far (beyaz)
- [ ] Arka stop lambası (kırmızı) — frene basınca yanmalı
- [ ] Dönüş sinyalleri (opsiyonel ama önerilen)
- [ ] Boynuz/korna çalışıyor
