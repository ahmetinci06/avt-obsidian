# 📜 TEKNOFEST Efficiency Challenge Kuralları
#yarış #kurallar #critical

> [!danger] Diskalifiye riski olan kurallar 🔴 ile işaretli

## Kategori: Elektromobil

### Yarış Formatı
- **Mesafe:** 22.5 km
- **Süre:** 29 dakika (max)
- **Sıralama:** Wh/km bazında (düşük = iyi)
- **Minimum ortalama hız:** 22.5/0.483 = ~46.5 km/h

### 🔴 Batarya Kuralları
- [ ] Maksimum batarya kapasitesi: **3 kWh**
- [ ] Batarya tipi: Li-ion, LiFePO4, veya benzeri
- [ ] BMS zorunlu
- [ ] Batarya kasası darbe dayanımlı

### 🔴 Yerlilik Şartları
- [ ] Motor sürücü (ESC) **yerli üretim** olmalı ← bizim AKS board!
- [ ] Yazılım yerli geliştirme
- [ ] Yerlilik beyanı hazırlanacak

### 🔴 Güvenlik Zorunlulukları
- [ ] 2× acil durdurma butonu (1 pilot, 1 dışarı)
- [ ] Rollbar (baş üstü min 5cm boşluk)
- [ ] Emniyet kemeri (min 4 nokta)
- [ ] Yangın söndürücü (ABC, min 1 kg)
- [ ] HV kablolar turuncu renk
- [ ] HV izolasyon > 500Ω/V
- [ ] Fren testi: her tekerlek kilitleyebilmeli
- [ ] Boynuz/korna çalışır durumda
- [ ] Ön/arka aydınlatma

### Teknik Kontrol
- [ ] Ağırlık tartımı
- [ ] Fren testi (eğimde durma)
- [ ] Elektrik izolasyon testi
- [ ] Güvenlik donanımı kontrolü
- [ ] Sürücü belgesi/lisans kontrolü
