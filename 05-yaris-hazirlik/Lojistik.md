# 🚚 Yarış Lojistiği
#yarış #lojistik

## Ulaşım
- [ ] Araç taşıma: römork / kapalı kamyonet
- [ ] Araç sabitleme kemerleri
- [ ] Taşıma sigortası (opsiyonel)
- [ ] Rota planlaması (İstanbul → yarış alanı)

## Konaklama
- [ ] Otel/pansiyon rezervasyonu (yarış haftası: 24-31 Ağu)
- [ ] Takım üyesi sayısı: ~22 kişi
- [ ] Yarış alanına yakın konaklama

## Götürülecek Ekipman

### Alet Çantası
- [ ] Tornavida seti (yıldız + düz)
- [ ] Alyan anahtar seti
- [ ] Somun anahtarı seti
- [ ] Pense, kargaburun, yan keski
- [ ] Multimetre (HV izolasyon testi için de)
- [ ] Osiloskop (opsiyonel ama çok faydalı)
- [ ] Lehim havyası + lehim + flux
- [ ] Isı tabancası + macun
- [ ] Kablo bağı (zip tie) bol miktarda
- [ ] Elektrik bandı + izole bant
- [ ] Silikon + yapıştırıcı

### Yedek Parçalar
- [ ] Yedek MCU board (veya en az 1 yedek AKS board)
- [ ] Yedek MOSFET'ler (2-3 adet)
- [ ] Yedek sigorta (çeşitli amper)
- [ ] Yedek kablo (çeşitli kesit)
- [ ] Yedek konnektör
- [ ] Yedek lastik (min 1)
- [ ] Fren sıvısı (küçük şişe)
- [ ] Cıvata/somun çeşitleri

### Elektronik
- [ ] Laptop (STM32CubeIDE kurulu, kodu derleyebilen)
- [ ] ST-LINK debugger
- [ ] USB kabloları
- [ ] Güç kaynağı (test için)
- [ ] Şarj cihazı (batarya için)

### Diğer
- [ ] Uzatma kablosu + çoklu priz
- [ ] Çadır/tente (güneş koruması)
- [ ] Masa + sandalye
- [ ] Su + atıştırmalık
- [ ] İlk yardım çantası
